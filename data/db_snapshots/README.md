## Description

Each `.snapshot` or `.tar` file contains a full collection backup including:

- Vector embeddings (e.g. 768-dim GTE multilingual model)
- Payload metadata (article title, section, source URL)
- Qdrant index structure and segment data

Example snapshot:

````

````

## Usage

To restore a collection:

```bash
curl -X POST \
  "http://localhost:6333/collections/ragx_documents_vDay/snapshots/upload" \
  -F "snapshot=@ragx_documents_vNewNEW-160262856051.snapshot"
````
